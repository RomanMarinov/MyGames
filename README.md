# MyGames

## Описание 
Андроид приложение позволяет 
- получать данные разных игр
- получать детализацию выбранных игр 

## Технологии 
Клиентская часть: android studio kotlin. Запрос api.

- clean,<br/>
- viewModel,<br/>
- mvvm, <br/>
- di hilt, <br/>
- navigation, <br/>
- databinding, <br/>
- animation start,<br/>
- animation fragment,<br/>
- recyclerViews,<br/>
- dialogs,<br/>
- retrofit<br/>
